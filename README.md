# スタジオゼブラ 予約管理システム

## プロジェクト概要

スタジオゼブラは新宿にある白ホリゾントを構えた撮影スタジオの予約管理システムです。このシステムでは、予約の半自動化、利用者への通知機能、リピーターユーザーへの利便性向上などを実現します。

### 主な機能

- ユーザー登録・認証
- 予約カレンダーの閲覧
- 仮予約・本予約の申し込み
- 予約の変更・キャンセルリクエスト
- 管理者による予約の承認・拒否
- Googleカレンダーとの連携
- 各種通知機能（将来拡張）

- **モバイル**: React Native（将来拡張）

## 開発計画

全6回のハンズオンセッション（各2時間×週1回）：

1. **プロジェクトキックオフ＆ゴール設定**
   - プロジェクト概要の確認
   - ステークホルダーと要件の洗い出し
   - MVPの明確化

2. **画面構成と情報設計**
   - 画面遷移図の作成
   - 情報アーキテクチャの検討
   - ユーザーストーリーの作成

3. **ワイヤーフレーム作成**
   - 主要画面のワイヤーフレーム設計
   - ユーザーフローの詳細化
   - インタラクションの検討

4. **UI モックアップ設計**
   - 高精細デザインの作成
   - スタイルガイドの策定
   - レスポンシブデザインの検討

5. **API 設計**
   - エンドポイントの詳細定義
   - データフォーマットの確定
   - セキュリティ設計

6. **DB 設計＆運用フロー検討**
   - データベース最終設計
   - バックアップ・リカバリ計画
   - 運用保守フローの確定

各セッションの成果物は Figma／OpenAPI 仕様書／ER 図などの形でリポジトリにまとめ、レビュー会を実施します。
